# ThinFilm-Vision 专业版自适应量测与诊断系统 V4.0
https://xumingzhao0401-blip.github.io/Thinfilm-Vision/
基于 Web 的工业级薄膜厚度光学量测工具。利用传递矩阵法 (TMM) 与 CIELAB 色彩约束，从反射率光谱中精确反演薄膜厚度，支持多层 Cauchy 色散模型、全局寻优防伪解、多线程并行计算以及晶圆级厚度分布可视化。

---

## ✨ 功能特性

- **高精度光学模型**  
  支持 1–5 层薄膜堆栈，每层使用 Cauchy 色散公式 `n(λ) = A + B/λ² + C/λ⁴` 描述材料折射率，可设定消光系数 `k`。内置标准 SiO₂ 材料库，可自定义保存/加载配方。

- **TMM 光谱引擎**  
  采用完整复数传递矩阵算法，在 380–780 nm 波长范围内以 1 nm 分辨率计算反射率，并与实测光谱进行全谱比对。

- **智能防伪解反演**  
  结合全局粗扫、局部 LM 精细迭代与 **色彩（ΔE）约束**，有效规避薄膜干涉的周期伪解，输出厚度及置信区间（±σ）。

- **晶圆级批量处理**  
  通过 CSV 导入多点坐标及对应光谱（支持稀疏 9 波段或密集 401 点全谱），利用 Web Worker 多线程并行拟合，实时反馈进度。

- **可视化诊断**  
  - 双轴诊断图表：同时显示理论光谱、实测光谱和拟合残差（右轴）。  
  - 2D Wafer Map：用 Viridis 色阶展示全晶圆厚度分布，自动计算均值、极差、Cpk。  
  - 0–1000 nm 连续色彩谱带：直观预览薄膜颜色随厚度的变化，用于工艺窗口定位。

- **工业 SPC 监控**  
  可设定规格上下限 (LSL/USL)、MSE 阈值、σ 上限，自动触发低置信度告警，并在 Wafer Map 上标记超规点。

- **纯前端离线运行**  
  除 Chart.js 外无其他依赖，下载后即可在本地浏览器中运行，无需服务器。

## 🧠 技术栈

| 模块 | 技术 |
|------|------|
| 光学计算 | 传递矩阵法 (TMM) · 复数运算 · Cauchy 色散 |
| 色彩科学 | D65 光源 · CIE 1931 2° 观察者 · Lab 颜色空间 |
| 数值优化 | 全局粗扫 + Levenberg-Marquardt 阻尼迭代 |
| 并行处理 | Web Worker 多线程 |
| 可视化 | Canvas · Chart.js 3.x |
| 数据导入 | CSV 文本解析 · 线性插值 |

## 🚀 快速开始

1. 克隆仓库或下载 `index.html` 文件。
2. **确保 Chart.js 可用**：本工具通过 CDN 引入 Chart.js，如需离线使用，请将 `https://cdn.jsdelivr.net/npm/chart.js` 替换为本地文件路径。
3. 在浏览器中打开 `index.html`（推荐 Chrome / Edge 最新版）。
4. 加载示例数据：点击 **“加载 17点 厂内样本”** 按钮，然后点击 **“启动多线程反演”** 即可看到拟合结果。

## 🎛️ 界面按钮说明

### 左侧控制面板
- **+ 新增常规膜**  
  添加一层新的薄膜（最多 5 层）。默认材质参数为 `A=2.0, B=0.01, C=0, k=0`，厚度 50 nm。

- **载入标准 SiO₂ 膜**  
  重置为单层 SiO₂ 热氧化层，Cauchy 系数 `A=1.458, B=0.0035, C=0, k=0`，初始厚度 300 nm。

- **激活拟合自变量** (单选按钮)  
  在堆栈面板中，勾选某一层的单选框，将其厚度作为反演目标变量。同一时间只能有一层为激活状态。

### 右侧操作面板
- **加载 17点 厂内样本**  
  自动生成一组模拟的 17 点晶圆光谱数据，用于测试系统功能。

- **⚡ 启动多线程反演**  
  对当前导入的 CSV 数据执行全局厚度反演。进度条会显示处理进度，完成后刷新诊断图和 Wafer Map。

- **💾 保存当前配方**  
  将当前堆栈配置（包括材料参数、拟合窗口、规格限等）导出为 JSON 文件，便于备份或分享。

- **📂 加载本地配方**  
  导入之前保存的 JSON 配方文件，恢复配置并清空现有数据。

### 通用功能
- **Wafer Map 点击交互**  
  在晶圆图上点击任意测试点，右侧诊断图会更新为该点的光谱拟合细节，厚度输入框也会同步。

- **厚度/参数手动调节**  
  在左侧堆栈面板中直接修改 Cauchy 系数或厚度数值，顶部色彩条和诊断图会实时刷新。

- **告警信息**  
  当拟合残差 (MSE) 或不确定度 (σ) 超出设定阈值，或厚度超出 LSL/USL 时，状态栏会显示红色警示。

## 🔧 使用注意事项

- 浏览器需支持 **Web Workers** 和 **ES6+** 语法（Chrome 68+、Firefox 79+、Edge 79+、Safari 14+）。
- 密集光谱导入时要求每行至少 404 列（点ID、X、Y + 380~780 nm 共 401 个反射率值）。
- 色彩约束阈值（ΔE）设置在 **系统监控与门限配置** 区域，默认 6.5，可根据工艺调整。
- 反演窗口应覆盖预期厚度的 ±30% 以上，避免因窗口过窄而遗漏真实解。
